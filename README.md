# Customer Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will churn based on various features like tenure, usage, and support interactions.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Multiple ML models (Logistic Regression, Random Forest, XGBoost)
- Model evaluation using ROC curve and confusion matrix

## 📈 Key Insights
- Customers with low engagement are more likely to churn
- Tenure plays a major role in retention
- Certain features strongly influence churn prediction

## 🚀 Future Improvements
- Deploy as a web app
- Improve model accuracy

## 📂 Dataset

- Dataset used: IBM Customer Churn Dataset  
- The dataset contains customer information such as demographics, account details, and service usage.  
- It is used to analyze customer behavior and predict churn (whether a customer will leave the service).
